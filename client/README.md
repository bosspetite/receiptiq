# ReceiptIQ client

Vite + React + Tailwind frontend. Full setup: **[project root README](../README.md)**.

## Your terminal says `...\receiptiq\client>`?

You are **one folder inside** the project. Either:

```powershell
cd ..
npm install
npm run dev
```

Or stay in `client` and run the whole app (API + Vite):

```powershell
npm run install-all
npm run dev:all
```

Frontend only (no Express API — dashboard/API calls will fail unless the server is running separately):

```powershell
npm run dev
```

Also read **`IF_YOU_ARE_HERE_README.txt`** in this folder.
